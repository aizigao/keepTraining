BLEACH》（原意“漂白”，片假名：ブリーチ）是久保带人创作，2001年开始连载于集英社旗下《周刊少年JUMP》上的漫画。亦改编有同名动画及轻小说。至日文单行本第71卷发售时，累计发行量突破8600万册。 [1] 台湾东立出版社译为《死神》（后为大陆网络所通用），香港文化传信译为《漂灵》，大陆连环画出版社先后拟定译名为“死神”、“净灵”，但均未通过新闻出版署审批，最后译为《境·界》。 简体中文单行本由株式会社集英社正式授权，连环画出版社引进出版，北京中少动漫图书有限公司代理发行简体中文版。2014年7月10日，《境·界》（死神）全彩版漫画登陆中国移动和动漫，这是集英社历史上第一次在中国授权全彩漫画电子版权 [2]。 2016年8月22日发行的《周刊少年JUMP》38号中完结。 中文名死神 原版名称BLEACH 别 名漂灵（香港）、境·界（大陆） 作 者久保带人 类 久保带人创作漫画的契机是因为画了露琪亚。先是设计好了这个角色，看了后觉得这就是死神，由此想到死神的故事。 最初考虑的死神的服装是西服，男性是黑色西装，女性是天主教会学校的制服。但当时在电影中已有穿黑色西装的死神。自古以来的死神是套着黑色的布的骷髅，对此进行改变而成的穿黑色西服的死神也已经有过了。那么还没有出现过的死神的样子到底是怎么样的呢。于是就想到黑色的和服。那么与和服搭配，所以武器就选择了刀。 斩魄刀是死神所使用的佩刀。其形状和能力，是以死神自身的灵魂为原型筑成的。死神通过知晓赋予自己的斩魄刀的名字，通过与之进行心灵的对话而得到力量。与死神一同降生，和死神一同消亡，这就是斩魄刀。 [10] 斩魄刀的创造者是零番队的“刀神”二枚屋王悦，他独自一人打造了所有的斩魄刀（除了黑崎一护的斩魄刀，二枚屋王悦在527话出，在同集中打造了一护的斩魄刀）。 零番队的“真名呼和尚”兵主部一兵卫，在所有“浅打”交到死神手里的那瞬间，就知道所有斩魄刀的名字。 鬼道 鬼道是《BLEACH》中只有死神才会使用的高级咒术，用以治疗，辅助攻击或牵制对手等，攻击型的鬼道称之为“破道”，级数越高其使用难度和威力也越高。防御、捕捉、追踪用的鬼道则称为“缚道”。治疗鬼道则被称作“回道”。为了施展鬼道，死神必须念出鬼道的名字和召唤的咒语（此步骤在动画中称为“吟唱”。在使用者灵力足够强的情况下，可免除吟唱）。不过治疗鬼道（回道）和某些特殊鬼道不需要念出咒语就能直接使用。某些能力高强的死神亦可以舍弃咏唱召唤的咒语，直接发出鬼道。 舍弃咏唱即是不用经过言灵程序就能使出鬼道，在战斗时机上有更好的发挥。不过会因应个人鬼道素质和鬼道的难度减弱一定鬼道的威力。后述咏唱就是专门施在以舍弃咏唱方式的鬼道上，加强舍弃咏唱后的鬼道威力。另外，也有非死神的灵体透过自修习得死神才会的鬼道，甚至是研发自己特有的鬼道（例如志波空鹤和志波岩鹫）。 虚化 浦原喜助在寻找突破死神的强度界线时所找到的答案“死神的虚化”，借由将这两个相反的存在分解，再重新组合得以追求更高一层的境界，或是将虚的魂魄灌进另外一个灵魂，借由破坏魂魄的界线，尝试将目标升华至更高层次的灵魂。但虚化意谓著灵魂将与自身化成的虚融合在一起，两者将成为无法分离的存在，而死神最终将必须面临被自身化成的虚所吞噬的命运。 随着虚化的恶化，原本的魂魄会变得和虚相同，成为失去理智的怪物。当魂魄间的界线、魂魄外界和外界的界线被破坏时，灵体就会自我消灭，称为“魂魄自杀”。 然而少数人借由“内在虚化抗争”成功战胜体内的虚之后，才将得以随心所欲的应用这股“虚化”的力量，而能够按照自己意识使用这股力量的人则被称作“假面”，因为当他们使用这股力量时脸上将戴上虚的面具，而散发出的灵压充满了不吉的压迫感。更高等的假面，甚至还能施展破面的归刃技能。目前到达此境界的只有黑崎一护、东仙要以及假面军势的众成员。其中东仙要以及假面军势则是经由蓝染以“崩玉”的力量一手创造的，只有黑崎一护是透过自然的“虚化”变化的。 失去力量的死神紧急时所利用的肉体。供需要长期留于现世的死神使用，穿上义骸后的死神可以让一般人看见他们。一般的义骸可以吸收大气里的灵子使死神的灵力逐渐恢复，但浦原特制给露琪亚的义骸却会逐渐削弱使用者的灵力，最后将使用者由死神变成凡人。为了确保死神的灵体与义骸协调，会需要特定的接着剂来固定灵魂与义骸，可是灵魂和义骸太过同调的话，会让死神在离开义骸的同时感到相当痛苦。后来浦原喜助甚至发明了可快速充气使用的义骸。此外，死神如果没有在现世穿戴义骸，甚至无法食用现世的食物和饮料。 至于浦原喜助给一心使用的义骸，则是用人类魂魄所制成，能将进驻的死神魂魄包覆为完全的人类，成为介于死神和人类间的存在，但是在使用义骸的期间将会无法使用死神之力，也无法看见虚。 种族划分 登场的角色来自不同的世界，可分作不同的生命体，每种生命体身内所含的“灵子”，浓度不同，因而力量也有所变化。已出现的灵体，如下列：生活在现世的人类、灭却师、魂魄、虚、死神、拥有死神化力量的虚—破面，以及拥有虚化力量的死神—假面，每位角色可按不同的类别来划分。 [11] 人类 “仍在世的人类”当中，已登场的角色全部居住于空座町。此地因为灵子浓度极高，居住在此的人亦容易拥有灵力，曾经被蓝染惣右介定为侵略目标，这里的角色性命危在旦夕。 某些人类拥有高度灵力，可以目击“整”、“虚”等魂魄，其中部分会协助死神的工作如消灭虚、引导整等（这种人类被称为代理死神），但有些人类会威胁到世界的安宁（如巴温特，完现术者）。 完现术者 某些具备虚的力量的特殊人类灵能力者。他们的亲人在他们出生前被虚袭击，虚的力量因此留在母亲身上，并且在他们出生时传到他们的身上，拥有这种能力的人，就被称作完现术者，他们的特有能力则被称为“完现术”（Fullbring）。 魂魄 “已死的人类”就是魂魄，魂魄会受死神指引而前往尸魂界。 在现世死去的人类灵魂，一旦为了某种原因依恋现世而不肯前行至安置魂魄的尸魂界，便成为孤魂野鬼，四处飘荡。“魂魄”通常不会危及人类，并会与能够看见他们的人对话。然而，“魂魄”由于内心深沉的渴望，假如未被及时指引往尸魂界，便可能成为恶灵，亦即“虚”。 死神 灵子浓度极高的生命体，拥有超过人类的能力，也可以与虚战斗。因为身体内灵子浓度高，同时也是虚最喜爱的食物。死神肩负着保卫人类，引导及净化“虚”、使之回到尸魂界的职责，维持两边世界的平衡，以免世界失稳而灭亡；并且是尸魂界的主宰。 虚 在现世的魂魄，过了一段时期，内心的欲望便会腐蚀身体，使之幻化成“虚”，亦即恶灵。虚会四处吞噬其他仍在生的人的灵魂，尤其是生前所爱的人。曾出现的虚，包括井上织姬的大哥、杀害小孩柴田勇一的连环杀手、六年前杀害黑崎一护母亲的虚等。一般在现世所见的虚，仍保有与能够看见他们的人对话的能力，亦即理智并非全灭。由于多数虚内心深处仍有良知，只要有其他灵体的协助，他们仍可过渡至尸魂界，等待转世。然而，大多数虚居于虚圏，力量及体型均远较流落现世的虚庞大，与死神及人类为敌，通称大虚。 大虚分为三级 下级大虚-基力安： 大部分虚为了弥补自己的灵力，会吞噬人类魂魄。而有的虚则会吞噬同类，吞噬了几百只后，成为没有特殊能力、体型巨大、智力低下、能够施放虚闪的大虚基力安。 并不能理解为几百只虚融合起来。 中级大虚-亚丘卡斯： 有些特别强大的虚在融合为基力安之后仍能保持原有的记忆、能力和智力，成为融合后基力安身体的主宰，这些少数的有意识体基力安不断吞噬其他的基力安生存，慢慢进化为智力、意识、能力和体型较为完全的中级大虚—亚丘卡斯。 上级大虚-瓦史托德： 大部分亚丘卡斯无论如何吞噬，也无法进化为瓦史托德，被同类伤害过的亚丘卡斯亦不能，极少数有资格进化的亚丘卡斯大量地吞噬其他虚，然后进化成上级大虚—瓦史托德，瓦史托德体型和智力与人类无异，广大空阔的虚圈中只有极少数的瓦史托德。 破面 Mundo”，处于现世与尸魂界之间的地带。原居民全部为虚，地面主要为一望无际的沙漠，沙丘中竖立着石英造成的树干。大气中灵子浓度非常高，沙丘中的小动物虚得以靠吸收灵子生存。原为拜勒岗所统治，但其后蓝染惣右介用斩魄刀“镜花水月”强使拜勒岗屈服，支配并重整虚夜宫。空座町大战后，蓝染败北被囚禁于地下监狱，原本身为蓝染部下的破面NO.3蒂雅·赫丽贝儿，便在战后的1年5个月期间担任虚圈临时统治者。后由于无形帝国为了网罗旗下兵力而进攻虚圈、对其他虚群展开屠杀的缘故，现时的虚圈已经成为无形帝国进攻尸魂界的根据地。 [10] 地狱 斩魄刀只能够洗清变成虚之后的罪孽，生前犯下重大犯罪的整、虚、破面等灵体将被导引至地狱，永世受到地狱的折磨。地狱门只出现于漫画12话（动画第5话），剧场版4的故事核心也是围绕着地狱。在剧场版4的设定里，地狱总共分为五层，还能让待在地狱里的灵魂获得无限复活的能力。但由于地狱的瘴气过于强烈，一旦地狱之门在没有将罪孽深重的魂魄拖入地狱的情况下，被其他灵体擅自打开，将会给现世带来相当严重的损害。 灵体共同名词 锁结和魄睡 灵体产生灵力的穴道，同时也是灵体的要害，若是被敌人击中这两处穴道，该灵体在半小时内就会死去，就算活下来也会力量尽失。死神被击中锁结和魄睡的话，就会失去死神的力量。 灵压 灵体的灵力，都以此称呼。每个灵体的灵压都是独一无二。灵压强弱，也会影响到战斗技能的威力、持久度、使用次数和灵压消耗量，灵压的移动与摇晃则被称作“魄动”。灵压会因为灵力全开、濒临危险或是受到激烈情绪的影响大幅高涨，也会因为施展技能、体力衰弱或是遭敌方吸收导致下降。不过灵压可借着休养或治疗技能（例如织姬的双天归盾和死神的治疗鬼道）来恢复，在肉身完全复原的状况下，灵压恢复过程也会比较顺利。死神的治疗原理则是先补充灵压，进而恢复身体创伤。此外，灵体经过的地方与碰触到的物品，会在短时间内残留少许的灵压。某些法术或物品也具备隐藏灵压的作用。同时灵体的灵压掌控能力，也会影响灵压感知能力的程度。 灵体之间战斗的时候，有时其中一方会刻意将自身的灵压隐藏到最低限度，令敌方无法感觉到对手的灵压，借此对敌方展开突击。若是某个灵体散发的灵压明显强过周遭的其他灵体，会让其他灵体受到震慑无法动弹。要是个体之间的灵压差距过大，灵压承受力较弱的那一方，甚至会无法承受过于强盛的灵压而当场丧命。一旦某个灵压特别强大的灵体，持续将自己的灵压保持于全开状态而不作任何收敛，将会因为自身散发的灵压超过承受极限，导致魂飞魄散。当灵体死亡的同时，灵压就会逐渐衰退并彻底消失。 灵压之芯 灵体具备的灵压核心，亦为维持灵体生命的能源，倘若灵压之芯离开灵体的身体，该灵体就会当场死亡。 灵子 组成个别灵体与灵界物质的基本单位。在灵界里死去的灵体，身体会分解成灵子，然后成为灵界物质的一部分。死神、虚、假面、破面、灭却师可将大气中的灵子凝聚在脚下，借此在空中移动（完现术者则是借着完现术吸取大气中的灵子，借此在空中短暂滞留）。在剧情后期，井上织姬经过虚圈的修行后，亦练就了将大气中的灵子凝聚在脚下，借此在空中移动的本事，成为剧中非属灭却师和完现术者却能在空中移动的人类的特例。 灵魂 每个灵体与物质皆具备的魂魄。在《BLEACH》的世界里，不论任何灵体、生物、物品，都有灵魂的存在。完现术者可借由抽取物质内的灵魂，衍生出不同的运用方式。 超越者 蓝染借由“崩玉”经过“魂魄组合交换”的数次进化后，对于自己的新力量所定义的境界，意指力量与能力已经“超越”虚与死神的范畴，成为不再是两者能够轻易干涉的更上位存在。而当完成“最后的月牙天冲”修练的一护与蓝染对战时，蓝染亦承认一护也曾靠着“最后的月牙天冲”到达与自己相同的境界。到达此境界的有黑崎一护与蓝染惣右介。 真血 于漫画第188章首度提及，疑似是某种死神的血脉（死神化的一心对Grand Fisher表明与一护之间的父子关系，Grand Fisher才导出此结论），但除此之外的相关细节暂时不详，而已知的“真血”只有黑崎一护。 家族编制 朽木家族：四大贵族之首，所有死神的典范，以维护与执行尸魂界的定则为职责。 四枫院家族：四大贵族之一，天赐兵装备，代代继承统领刑军的职责。 志波家族：曾经的五大贵族之一，已经没落。 京乐家族：上级贵族家族 浮竹家族：下级贵族家族
